	commands that can be used to define the DB schema.It is used to create and modify the structure of DB objects. it manipulate the DB structure not the data itself.

- CREATE : Used to create DB or object such as tables, etc.
- DROP: It is used to delete an existing DB or an object within a DB.
- ALTER: It is used to modify the structure of an already existing table.
- TRUNCATE: It is used to remove all the records from a table not the table but the records inside the table.
